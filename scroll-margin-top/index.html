scroll-margin-top

Anchor links no longer hide behind your sticky navbar. One property fixes the offset — no JavaScript, no scroll padding hacks.

Live Demo
JUMP TO →
DEVIGNITOR.COM sticky nav — 44px height
Section 01
Introduction to CSS Layouts
Without scroll-margin-top, this heading would slide under the sticky navigation bar when jumped to via anchor link. Notice it lands perfectly — 44px below the nav, exactly right.
Section 02
Flexbox & Grid Fundamentals
The scroll-margin-top value should match your sticky nav height. Here it's set to 44px — the exact height of the dark navigation bar above. Clean, precise, predictable.
Section 03
Advanced Positioning
This technique works with any scroll container, not just the root viewport. Set it once per element class and forget about it. Works with smooth scroll and instant scroll alike.
Property
scroll-margin-top: 44px
Use case
Sticky nav offset for anchor links
