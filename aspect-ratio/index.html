aspect-ratio

Lock any element to a perfect ratio — cards, videos, avatars — without padding hacks. One property. The browser handles the math.

Live Demo
Common ratios — resize window to see them hold

4:3 card · photo
Old Hack Padding-top trick 
.video {
  position: relative;
  padding-top: 56.25%;
  /* 9/16 = 0.5625 */
}
.video iframe {
  position: absolute;
  inset: 0;
}
Modern aspect-ratio 
.video {
  width: 100%;
  aspect-ratio: 16 / 9;
  /* That's it. Done. */
}

/* No extra wrappers
   No positioning needed */
